# Archie Viterbo Portfolio

Interactive portfolio website for Archie D. Viterbo, built with React, TypeScript, Vite, and Tailwind CSS.

## Overview

This project presents a cyber-style portfolio with animated sections, project showcases, resume details, and a contact console.

## Features

- Responsive portfolio layout
- Animated hero and background effects
- Project and resume sections
- Contact and admin panels
- Built with Vite for fast local development

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Start the development server:
   `npm run dev`

## Build

To create a production build:

```bash
npm run build
```
